## 1. 회귀 분석 개요

### 1.1 회귀 분석이란?

회귀 분석은 **종속 변수**(반응 변수, 결과 변수)와 하나 이상의 **독립 변수**(예측 변수, 특성 변수) 사이의 관계를 모형화하는 통계적 방법이다.

회귀 분석의 목적은 다음과 같다:
- **설명적 모형화**: 예측 변수가 반응 변수에 어떤 영향을 미치는지 이해
- **예측적 모형화**: 반응 변수의 미래 값을 예측

### 1.2 회귀 분석의 유형

| 유형 | 반응 변수 | 예측 변수 | 예시 |
|------|-----------|-----------|------|
| 단순 선형 | 연속형 | 하나의 연속형 | 매출 ~ TV 광고비 |
| 다중 선형 | 연속형 | 다수 | 가격 ~ 연식 + 주행거리 + 마력 |
| 로지스틱 | 이진형 (0/1) | 하나 이상 | 질병 ~ 위험 요인 |
| 다항식 | 연속형 | 비선형 항 | 성장 ~ 시간 + 시간^2 |

## 2. 단순 선형 회귀

### 2.1 모형

단순 선형 회귀는 $X$와 $Y$ 사이의 선형 관계를 가정한다:

$$Y \approx \beta_0 + \beta_1 X$$

여기서:
- $\beta_0$는 **절편**($X = 0$일 때 $Y$의 기대값)
- $\beta_1$은 **기울기**($X$가 1단위 증가할 때 $Y$의 평균 변화량)
- $\epsilon$은 **오차항**(잡음 및 모형에 포함되지 않은 효과를 포착)

완전한 통계 모형은 다음과 같다:

$$Y = \beta_0 + \beta_1 X + \epsilon$$

### 2.2 계수 추정(최소제곱법)

최소제곱법은 **잔차제곱합(RSS)**을 최소화하는 $\hat{\beta}_0$와 $\hat{\beta}_1$을 구한다:

$$RSS = \sum_{i=1}^{n} e_i^2 = \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$$

여기서 $e_i = y_i - \hat{y}_i$는 **잔차**(관측값과 예측값의 차이)이다.

닫힌 형태의 해는 다음과 같다:

$$\hat{\beta}_1 = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2}$$

$$\hat{\beta}_0 = \bar{y} - \hat{\beta}_1 \bar{x}$$

### 2.3 R에서의 구현

```r
# 단순 선형 회귀 적합
model <- lm(y ~ x, data = mydata)
summary(model)
```

> **핵심 포인트:** R의 `lm()` 함수는 `y ~ x` 형태의 수식 표기법을 사용하며, `y`는 반응 변수, `x`는 예측 변수이다. `~` 기호는 "~의 함수로 모형화된다"를 의미한다.

## 3. 다중 선형 회귀

### 3.1 모형

다중 선형 회귀는 단순 선형 회귀를 여러 예측 변수로 확장한 것이다:

$$Y = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_p X_p + \epsilon$$

### 3.2 예제: Toyota Corolla 가격 예측

중고차 가격을 속성에 기반하여 예측하는 전형적인 예제이다:

| 변수 | 설명 |
|------|------|
| Price | 제시 가격(유로, 반응 변수) |
| Age | 연식(월 단위) |
| Kilometers | 주행거리(km) |
| Fuel Type | 연료 유형(가솔린, 디젤, CNG) |
| HP | 마력 |
| Metallic | 메탈릭 색상 여부(Yes=1, No=0) |
| Automatic | 자동 변속기 여부(Yes=1, No=0) |
| CC | 실린더 용량(cc) |
| Doors | 문 수 |
| QuartTax | 분기별 도로세(유로) |
| Weight | 중량(kg) |

### 3.3 데이터 분할

모형을 적합하기 전에 데이터를 **훈련** 세트와 **검증** 세트로 분할한다:

```r
# 데이터 분할: 60% 훈련, 40% 검증
set.seed(1)
train.index <- sample(c(1:1000), 600)
train.df <- car.df[train.index, selected.var]
valid.df <- car.df[-train.index, selected.var]
```

### 3.4 R에서의 적합

```r
# 나머지 모든 열을 예측 변수로 사용 (.은 "나머지 모든 변수"를 의미)
car.lm <- lm(Price ~ ., data = train.df)
options(scipen = 999)   # 과학적 표기법 비활성화
summary(car.lm)
```

## 4. 회귀 방정식 추정

### 4.1 출력 해석

`summary()` 출력에는 다음이 포함된다:

| 구성 요소 | 설명 |
|-----------|------|
| **Estimate** | 추정 계수 값 ($\hat{\beta}_i$) |
| **Std. Error** | 추정값의 표준 오차 |
| **t value** | 검정 통계량: $t_i = \hat{\beta}_i / se(\hat{\beta}_i)$ |
| **Pr(>\|t\|)** | 양측 가설 검정의 p값 |
| **Multiple R-squared** | $R^2$ — 설명된 분산의 비율 |
| **Adjusted R-squared** | $R^2_{adj}$ — 예측 변수 수를 고려하여 조정 |
| **F-statistic** | 모형의 전체적 유의성 검정 |

### 4.2 유의성 코드

```
Signif. codes:  0 '***' 0.001 '**' 0.01 '*' 0.05 '.' 0.1 ' ' 1
```

p값이 작을수록(< 0.05) 해당 예측 변수가 **통계적으로 유의미**함을 나타낸다.

### 4.3 결정 계수($R^2$)

결정 계수는 모형이 데이터를 얼마나 잘 적합하는지를 측정한다:

$$R^2 = 1 - \frac{RSS}{TSS} = 1 - \frac{\sum(y_i - \hat{y}_i)^2}{\sum(y_i - \bar{y})^2}$$

- $R^2 = 1$: 완벽한 적합
- $R^2 = 0$: 모형이 아무것도 설명하지 못함

**수정된** $R^2$는 불필요한 예측 변수의 추가에 대해 벌점을 부여한다:

$$R^2_{adj} = 1 - \frac{(1 - R^2)(n - 1)}{n - p - 1}$$

> **핵심 포인트:** 예측 변수의 수가 다른 모형을 비교할 때는 항상 수정된 $R^2$를 사용해야 한다. 변수를 더 추가하면 $R^2$는 항상 증가하지만, 모형이 반드시 개선되는 것은 아니다.

## 5. 모형 평가 지표

### 5.1 예측 및 잔차

```r
# 검증 세트에 대한 예측
car.lm.pred <- predict(car.lm, valid.df)

# 잔차 계산
some.residuals <- valid.df$Price[1:20] - car.lm.pred[1:20]
data.frame("Predicted" = car.lm.pred[1:20],
           "Actual" = valid.df$Price[1:20],
           "Residual" = some.residuals)
```

### 5.2 주요 정확도 지표

| 지표 | 수식 | 해석 |
|------|------|------|
| **MAE** | $\frac{1}{n}\sum\|e_i\|$ | 평균 절대 오차의 크기 |
| **ME** | $\frac{1}{n}\sum e_i$ | 평균 오차(편향 방향 표시) |
| **MPE** | $100 \times \frac{1}{n}\sum \frac{e_i}{y_i}$ | 방향이 있는 백분율 오차 |
| **MAPE** | $100 \times \frac{1}{n}\sum \frac{\|e_i\|}{y_i}$ | 평균 백분율 편차 |
| **RMSE** | $\sqrt{\frac{1}{n}\sum e_i^2}$ | 평균 제곱근 오차(Y와 동일한 단위) |

```r
library(forecast)
accuracy(car.lm.pred, valid.df$Price)
#         ME     RMSE      MAE       MPE     MAPE
# 19.62  1325.24  1048.60  -0.75    9.35
```

> **핵심 포인트:** RMSE는 반응 변수와 동일한 단위를 가지며, 큰 오차에 대해 MAE보다 더 큰 벌점을 부여하기 때문에 가장 널리 사용되는 지표이다.

## 6. 변수 선택

### 6.1 변수 선택의 필요성

모든 예측 변수가 모형에 의미 있는 기여를 하는 것은 아니다. 불필요한 변수를 포함하면 다음과 같은 문제가 발생한다:
- 정확도 개선 없이 모형 복잡도만 증가
- **과적합**(모형이 훈련 데이터를 암기하지만 새로운 데이터에서는 실패)
- 해석 가능성 저하

### 6.2 방법론

| 방법 | 설명 |
|------|------|
| **전진 선택(Forward Selection)** | 빈 모형에서 시작하여 변수를 하나씩 추가 |
| **후진 제거(Backward Elimination)** | 모든 변수에서 시작하여 하나씩 제거 |
| **단계적 선택(Stepwise)** | 전진과 후진의 조합 |
| **최적 부분 집합(Best Subset)** | 가능한 모든 조합을 평가 |

### 6.3 R에서의 구현

```r
# AIC 기준을 사용한 단계적 회귀
step(car.lm, direction = "both")

# 후진 제거
step(car.lm, direction = "backward")

# 전진 선택
step(lm(Price ~ 1, data = train.df),
     scope = formula(car.lm),
     direction = "forward")
```

## 7. R에서의 지리 데이터 회귀

### 7.1 미국 문해율 단계구분도와 회귀 분석

과제 2는 지리적 시각화와 통계적 분석을 결합한다:

```r
library(tidyverse)
library(sf)

# 미국 주 shapefile 로드
USMap <- st_read("data/cb_2024_us_state_5m.shp")

# 문해율 데이터 로드
litRates <- read.csv("data/us_literacy_rates_by_state_2025.csv")

# 공간 데이터와 속성 데이터 결합
continentalUS <- USMap %>%
  left_join(litRates, by = c("NAME" = "State")) %>%
  filter(NAME != "Hawaii" & NAME != "Alaska" &
         NAME != "Puerto Rico" & !is.na(Rate))

# 단계구분도 생성
ggplot(continentalUS, aes(geometry = geometry, fill = Rate)) +
  geom_sf() +
  coord_sf()
```

### 7.2 공간 데이터 개념

| 개념 | 설명 |
|------|------|
| Shapefile | 지리적 피처를 저장하는 벡터 형식(.shp, .dbf, .shx, .prj) |
| CRS | 좌표 참조 시스템(예: WGS84의 경우 EPSG:4326) |
| `sf` | R에서 공간 데이터를 다루는 Simple Features 패키지 |
| `st_read()` | Shapefile을 R의 sf 객체로 읽어들임 |
| `geom_sf()` | 공간 데이터를 위한 ggplot2 지오메트리 |
