## 1. 동기: 분류 vs 회귀

### 1.1 분류 문제에 선형 회귀를 사용할 수 없는 이유

선형 회귀는 연속형 반응 변수 $Y \in (-\infty, +\infty)$를 예측한다. 반응 변수가 **이진형**(예: Yes/No, 0/1)일 때 선형 회귀는 다음과 같은 치명적 한계를 갖는다:

- 예측값이 $[0, 1]$ 범위 **밖**으로 벗어날 수 있어 확률로 해석할 수 없음
- 선형 모형은 예측 변수의 일정한 효과를 가정하는데, 이는 경계가 있는 결과에는 비현실적
- 잔차가 이분산적이고 비정규적이어서 OLS 가정을 위반

### 1.2 분류 문제의 설정

| 측면 | 회귀 | 분류 |
|------|------|------|
| 반응 변수 | 연속형 ($Y \in \mathbb{R}$) | 범주형(이진 또는 다중 클래스) |
| 목표 | 수치 값 예측 | 클래스 레이블 예측 |
| 출력 | 점 추정값 | 클래스 소속 확률 |
| 예시 | 주택 가격 예측 | 질병 진단(환자 / 건강) |

로지스틱 회귀는 **회귀 프레임워크**를 사용하여 **분류 문제**를 해결하는 가교 역할을 한다. 이는 $Y$가 특정 클래스에 속할 확률을 모형화한다.

## 2. 로지스틱 함수(시그모이드)

### 2.1 정의

**로지스틱 함수**(시그모이드 함수라고도 함)는 임의의 실수 입력을 구간 $(0, 1)$로 매핑한다:

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

여기서 $z = \beta_0 + \beta_1 X_1 + \beta_2 X_2 + \cdots + \beta_p X_p$는 예측 변수의 선형 결합이다.

### 2.2 성질

| 성질 | 설명 |
|------|------|
| 치역 | $(0, 1)$ — 항상 유효한 확률을 출력 |
| 단조성 | 순증가 |
| 대칭성 | $\sigma(-z) = 1 - \sigma(z)$ |
| 중점 | $\sigma(0) = 0.5$ |
| 도함수 | $\sigma'(z) = \sigma(z)(1 - \sigma(z))$ |

### 2.3 로지스틱 회귀 모형

예측 변수가 주어졌을 때 반응 변수가 1일 확률은 다음과 같다:

$$P(Y = 1 \mid X) = \frac{1}{1 + e^{-(\beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p)}}$$

이를 통해 예측 변수 값에 관계없이 예측 확률이 항상 0과 1 사이에 있도록 보장한다.

## 3. 오즈와 로그 오즈

### 3.1 오즈(Odds)

사건의 **오즈**는 사건이 발생할 확률과 발생하지 않을 확률의 비율로 정의된다:

$$\text{Odds} = \frac{P(Y=1)}{P(Y=0)} = \frac{P(Y=1)}{1 - P(Y=1)}$$

- $P = 0.8$이면 $\text{Odds} = 0.8 / 0.2 = 4$ (사건이 발생하지 않을 때보다 4배 더 발생할 가능성이 높음)
- 오즈의 범위는 $0$에서 $+\infty$

### 3.2 로그 오즈(로짓)

오즈의 자연 로그를 취하면 **로짓(logit)** 함수가 된다:

$$\text{logit}(P) = \log\left(\frac{P}{1-P}\right) = \beta_0 + \beta_1 X_1 + \cdots + \beta_p X_p$$

이것이 핵심 변환이다: $P$는 0과 1 사이에 제한되지만, 로그 오즈는 $(-\infty, +\infty)$로 **제한이 없어** 예측 변수의 선형 함수로 모형화할 수 있다.

### 3.3 관계 요약

$$X \xrightarrow{\text{선형 결합}} z = \beta_0 + \beta X \xrightarrow{\text{시그모이드}} P = \sigma(z) \xrightarrow{\text{임계값}} \hat{Y} \in \{0, 1\}$$

## 4. 최대 우도 추정

### 4.1 최소제곱법을 사용할 수 없는 이유

선형 회귀와 달리 로지스틱 회귀에서는 최소제곱법을 **사용할 수 없다**. 그 이유는:
- 반응 변수가 이진형이므로 연속형이 아님
- 예측 변수와 $P(Y=1)$ 사이의 관계가 비선형
- 최소제곱법이 확률 모형에 대해 의미 있는 추정을 생성하지 않음

### 4.2 우도 함수

$n$개의 관측값에 대해 **우도**는 모형 매개변수가 주어졌을 때 실제 데이터를 관측할 확률이다:

$$L(\beta) = \prod_{i=1}^{n} P(Y_i = y_i \mid X_i) = \prod_{i=1}^{n} p_i^{y_i}(1 - p_i)^{1 - y_i}$$

여기서 $p_i = P(Y_i = 1 \mid X_i)$이다.

### 4.3 로그 우도

로그를 취하면 곱이 합으로 변환되어 최적화가 용이해진다:

$$\ell(\beta) = \sum_{i=1}^{n} \left[ y_i \log(p_i) + (1 - y_i) \log(1 - p_i) \right]$$

최대 우도 추정(MLE)은 $\ell(\beta)$를 **최대화**하는 $\hat{\beta}$를 구한다. 이는 **Newton-Raphson** 또는 **반복 가중 최소제곱(IRLS)** 등의 알고리즘을 사용하여 반복적으로 풀이한다.

### 4.4 R에서의 구현

```r
# 최대 우도를 사용한 로지스틱 회귀 적합
model <- glm(y ~ x1 + x2, data = mydata, family = binomial)
summary(model)
```

> **핵심 포인트:** `family = binomial`이 지정된 `glm()` 함수는 내부적으로 MLE를 사용한다. `summary()` 출력은 **로그 오즈** 척도에서 계수를 보고한다.

## 5. 모형 해석

### 5.1 계수 해석

로지스틱 회귀에서 계수는 예측 변수가 1단위 증가할 때 **로그 오즈**의 변화를 나타낸다:

$$\log\left(\frac{P}{1-P}\right) = \beta_0 + \beta_1 X_1$$

- $\beta_0$: 모든 예측 변수가 0일 때 $Y=1$의 로그 오즈
- $\beta_k$: 다른 변수를 고정했을 때, $X_k$가 1단위 증가하면 로그 오즈가 $\beta_k$만큼 변함

### 5.2 오즈비(Odds Ratio)

계수를 지수화하면 **오즈비**를 얻는다:

$$\text{OR}_k = e^{\beta_k}$$

| $\beta_k$ | $e^{\beta_k}$ | 해석 |
|-----------|---------------|------|
| 양수 | $> 1$ | 예측 변수가 $Y=1$의 오즈를 **증가**시킴 |
| 0 | $= 1$ | 예측 변수가 **효과 없음** |
| 음수 | $< 1$ | 예측 변수가 $Y=1$의 오즈를 **감소**시킴 |

예를 들어, $\beta_1 = 0.693$이면 $e^{0.693} = 2.0$이므로, $X_1$이 1단위 증가할 때 $Y=1$의 오즈가 **2배**가 된다.

### 5.3 R에서의 오즈비 계산

```r
# 95% 신뢰구간과 함께 오즈비 구하기
exp(coef(model))
exp(confint(model))
```

## 6. 모형 평가

### 6.1 예측 클래스

**결정 임계값**(일반적으로 0.5)이 예측 확률을 클래스 레이블로 변환한다:

$$\hat{Y} = \begin{cases} 1 & \text{if } P(Y=1 \mid X) \geq 0.5 \\ 0 & \text{otherwise} \end{cases}$$

### 6.2 혼동 행렬(Confusion Matrix)

혼동 행렬은 분류 성능을 요약한다:

|  | 예측 양성 | 예측 음성 |
|--|-----------|-----------|
| **실제 양성** | 참 양성 (TP) | 거짓 음성 (FN) |
| **실제 음성** | 거짓 양성 (FP) | 참 음성 (TN) |

혼동 행렬에서 도출되는 주요 지표:

| 지표 | 수식 | 설명 |
|------|------|------|
| **정확도(Accuracy)** | $\frac{TP + TN}{TP + TN + FP + FN}$ | 전체 정답 비율 |
| **민감도(Sensitivity, Recall)** | $\frac{TP}{TP + FN}$ | 참 양성 비율 |
| **특이도(Specificity)** | $\frac{TN}{TN + FP}$ | 참 음성 비율 |
| **정밀도(Precision)** | $\frac{TP}{TP + FP}$ | 양성 예측 정확도 |
| **F1 점수** | $\frac{2 \cdot \text{Precision} \cdot \text{Recall}}{\text{Precision} + \text{Recall}}$ | 정밀도와 재현율의 조화 평균 |

### 6.3 ROC 곡선과 AUC

**수신자 조작 특성(ROC)** 곡선은 다양한 임계값에서 참 양성 비율(민감도)과 거짓 양성 비율(1 - 특이도)을 도시한다.

- 완벽한 분류기는 좌상단 모서리를 따른다 (TPR = 1, FPR = 0)
- 무작위 분류기는 대각선을 따른다
- **곡선 아래 면적(AUC)**은 모형의 전반적 성능을 정량화한다:

| AUC | 모형 품질 |
|-----|-----------|
| 0.9--1.0 | 우수 |
| 0.8--0.9 | 양호 |
| 0.7--0.8 | 보통 |
| 0.6--0.7 | 미흡 |
| 0.5 | 판별 불가(무작위) |

### 6.4 R에서의 ROC 곡선

```r
library(pROC)
roc_obj <- roc(actual, predicted_prob)
plot(roc_obj, main = "ROC Curve", col = "blue", lwd = 2)
auc(roc_obj)
```

## 7. 다항 로지스틱 회귀

### 7.1 다중 클래스로의 확장

반응 변수가 두 개 이상의 범주를 가질 때($K > 2$), **다항 로지스틱 회귀**가 이진 모형을 일반화한다. 하나의 범주가 **참조**(기준) 범주로 지정되며, 모형은 $K - 1$개의 계수 세트를 추정한다.

참조 범주 $K$에 대한 범주 $k$의 경우:

$$\log\left(\frac{P(Y = k)}{P(Y = K)}\right) = \beta_{k0} + \beta_{k1} X_1 + \cdots + \beta_{kp} X_p$$

### 7.2 예측 확률

각 클래스의 확률은 **소프트맥스(softmax)** 함수를 사용하여 계산된다:

$$P(Y = k \mid X) = \frac{e^{z_k}}{\sum_{j=1}^{K} e^{z_j}}$$

여기서 $z_k = \beta_{k0} + \beta_{k1} X_1 + \cdots + \beta_{kp} X_p$이고 $z_K = 0$(참조 범주)이다.

### 7.3 R에서의 구현

```r
library(nnet)
multi_model <- multinom(Species ~ ., data = iris)
summary(multi_model)

# 예측 확률
predict(multi_model, newdata = iris[1:5, ], type = "probs")
```

> **핵심 포인트:** `nnet` 패키지의 `multinom()` 함수는 다항 로지스틱 회귀를 적합한다. 참조 범주가 아닌 각 클래스는 자체적인 계수 세트를 갖는다.
