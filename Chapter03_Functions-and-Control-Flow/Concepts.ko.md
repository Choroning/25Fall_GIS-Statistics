## 1. 변수와 함수

### 1.1 기본 변수 할당

```r
x <- 10        # 수치형 값 할당
y <- "HI"      # 문자형 값 할당
```

### 1.2 내장 함수

R은 일반적인 연산을 위한 많은 내장 함수를 제공한다:

```r
print("Hello World")   # 콘솔에 값 출력
sum(1:100)              # 1부터 100까지 정수의 합 -> 5050
Sys.Date()              # 현재 시스템 날짜 반환
```

> **핵심 요점:** `sum()`, `mean()`, `min()`, `max()` 등의 함수는 R의 `base` 패키지에 포함되어 있어 별도의 라이브러리를 로드하지 않아도 항상 사용할 수 있다.

## 2. 출력 함수

### 2.1 print() vs. cat()

| 함수 | 동작 |
|------|------|
| `print()` | 한 번에 하나의 객체를 출력하며, 유형 서식이 포함됨 |
| `cat()` | 여러 객체를 연결하여 출력하며, 일반 텍스트로 출력됨 |

```r
x <- c(1, 2, 3)
y <- "a"

# print()는 한 번에 하나의 객체만 처리 가능
print(x)              # [1] 1 2 3
# print(x, y)         # 오류: invalid printing digits
print(x); print(y)    # 작동: 각각 별도로 출력

# cat()은 여러 객체를 연결하여 출력
cat(x, y)             # 1 2 3 a (서식 없는 일반 텍스트)
cat("Value:", x, "\n") # 줄바꿈 문자 지원
```

> **핵심 요점:** 단일 객체의 서식화된 출력에는 `print()`를 사용한다. 여러 값을 결합하거나 `\n`(줄바꿈) 등 사용자 정의 서식이 필요할 때는 `cat()`을 사용한다.

## 3. 사용자 정의 함수

### 3.1 함수 문법

사용자 정의 함수는 네 가지 구성 요소를 갖는다:

1. **함수 이름**: R 환경에 저장되며 함수를 호출할 때 사용
2. **매개변수**: 함수 호출 시 전달되는 입력 변수
3. **함수 본문**: 함수의 로직을 구현하는 코드
4. **반환 값**: 함수가 생성하는 결과

```r
# 일반 문법
function_name <- function(param1, param2, ...) {
  # 함수 본문
  ...
  return(result)
}
```

### 3.2 예제: 곱셈 함수

```r
multi_f <- function(x, y, z) {
  res <- x * y * z
  return(res)
}

multi_f(3, 5, 6)   # 90을 반환

a <- multi_f(3, 5, 6)
a                    # 90
```

### 3.3 예제: 정규화(Rescale) 함수

값을 [0, 1] 범위로 정규화하는 실용적인 예제:

```r
rescale <- function(x) {
  rng <- range(x, na.rm = TRUE)
  cat("The range is ", rng, "\n")
  (x - rng[1]) / (rng[2] - rng[1])
}

rescale(c(0, 5, 10))               # 0.0 0.5 1.0
rescale(c(0, 5, 10, NA, NA))       # 0.0 0.5 1.0 NA NA
```

### 3.4 함수를 작성해야 할 때

*R for Data Science*에 나오는 "세 번 규칙(Rule of Three)"에 따르면:

> 코드 블록을 두 번 이상 복사하여 붙여넣었다면 함수를 작성하는 것을 고려해야 한다.

함수를 만드는 세 가지 핵심 단계:
1. 함수에 설명적인 **이름**을 지정한다
2. `function()` 안에 **인수**(입력)를 나열한다
3. 작동하는 코드를 **본문**(`{ }` 블록)에 배치한다

> **핵심 요점:** 먼저 작동하는 코드를 작성한 후 이를 함수로 변환하는 것이, 함수를 먼저 만들고 작동시키려는 것보다 쉽다.

## 5. 티블

### 5.1 티블 vs. 데이터 프레임

티블(Tibble)은 `tibble` 패키지에서 제공하는 데이터 프레임의 현대적 재해석이다. `tidyverse` 생태계의 일부이다.

```r
# 정규분포 난수 데이터로 티블 생성
df <- tibble::tibble(
  a = rnorm(10),
  b = rnorm(10),
  c = rnorm(10),
  d = rnorm(10)
)
df
```

### 5.2 수동 정규화 (함수 사용 없이)

함수의 필요성을 보여주는 반복적인 코드:

```r
# 각 열을 [0, 1] 범위로 정규화 (장황한 접근 방식)
df$a <- (df$a - min(df$a, na.rm = TRUE)) /
  (max(df$a, na.rm = TRUE) - min(df$a, na.rm = TRUE))
df$b <- (df$b - min(df$b, na.rm = TRUE)) /
  (max(df$b, na.rm = TRUE) - min(df$b, na.rm = TRUE))
# ... 각 열에 대해 반복
```

이러한 반복적인 패턴이 바로 `rescale()` 함수를 만들어야 하는 이유이다.

### 5.3 NA 값 처리

```r
# NA는 R에서 결측값을 나타냄
a2 <- c(5, 2, 100, 25, NA, 6)
min(a2)                   # NA (NA와의 모든 연산은 NA를 반환)
min(a2, na.rm = TRUE)     # 2 (na.rm = TRUE는 NA 값을 무시)
```

> **핵심 요점:** 결측값(`NA`)이 포함될 수 있는 데이터에 대해 요약 통계를 계산할 때는 항상 `na.rm = TRUE`를 사용해야 한다.

## 6. 연산자

### 6.1 산술 연산자

| 연산자 | 연산 | 예시 |
|--------|------|------|
| `+` | 덧셈 | `5 + 3` -> `8` |
| `-` | 뺄셈 | `10 - 4` -> `6` |
| `*` | 곱셈 | `5 * 6` -> `30` |
| `/` | 나눗셈 | `20 / 4` -> `5` |
| `%/%` | 정수 나눗셈 | `20 %/% 7` -> `2` |
| `%%` | 나머지 연산 | `20 %% 7` -> `6` |
| `^` 또는 `**` | 거듭제곱 | `6 ^ 2` -> `36` |

### 6.2 관계(비교) 연산자

| 연산자 | 의미 | 예시 | 결과 |
|--------|------|------|------|
| `>` | 초과 | `5 > 3` | `TRUE` |
| `>=` | 이상 | `5 >= 6` | `FALSE` |
| `<` | 미만 | `5 < 3` | `FALSE` |
| `<=` | 이하 | `5 <= 6` | `TRUE` |
| `==` | 같음 | `5 == 5` | `TRUE` |
| `!=` | 같지 않음 | `5 != 5` | `FALSE` |
| `!` | 논리 NOT | `!5` | `FALSE` |

### 6.3 논리 연산자

```r
x <- 1:3
y <- 3:1

(x > 0) & (y > 1)   # 원소별 AND: TRUE TRUE FALSE
(x > 0) | (y > 1)   # 원소별 OR:  TRUE TRUE TRUE
```

### 6.4 벡터화된 비교

```r
a <- c(1, 5, 10)
a > 3    # FALSE TRUE TRUE (원소별 비교)
```

## 7. 조건문

### 7.1 if-else 문법

```r
if (condition) {
  # 조건이 TRUE일 때 실행되는 코드
} else {
  # 조건이 FALSE일 때 실행되는 코드
}
```

### 7.2 예제: 이차방정식 풀이

```r
# a2*x^2 + a1*x + a0 = 0의 근 구하기
rm(list = ls())

# 계수 입력
a2 <- 1
a1 <- 4
a0 <- 0

# 판별식 계산
discrim <- a1^2 - 4 * a2 * a0

# 판별식 값에 따라 근 계산
if (discrim > 0) {
  roots <- c(
    (-a1 + sqrt(a1^2 - 4*a2*a0)) / (2*a2),
    (-a1 - sqrt(a1^2 - 4*a2*a0)) / (2*a2)
  )
} else if (discrim == 0) {
  roots <- -a1 / (2 * a2)
} else {
  roots <- c()   # 실근 없음
}
show(roots)
```

> **핵심 요점:** R에서 여는 중괄호 `{`는 반드시 `if`/`else` 키워드와 같은 줄에 위치해야 한다. `else`도 이전 블록의 닫는 중괄호 `}`와 같은 줄에 있어야 한다.

## 8. 반복문

### 8.1 for 반복문

```r
x_list <- seq(1, 9, by = 2)   # 1, 3, 5, 7, 9

sum_x <- 0
for (x in x_list) {
  sum_x <- sum_x + x
  cat("The current loop element is ", x, "\n")
  cat("The cumulative total is ", sum_x, "\n")
}
```

**출력:**
```
The current loop element is  1
The cumulative total is  1
The current loop element is  3
The cumulative total is  4
...
The current loop element is  9
The cumulative total is  25
```

### 8.2 while 반복문

```r
counter <- 1
while (counter <= 5) {
  cat("Counter:", counter, "\n")
  counter <- counter + 1
}
```
