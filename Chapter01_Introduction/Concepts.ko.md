## 1. R이란?

### 1.1 정의와 역사

R은 통계 컴퓨팅과 그래픽을 위해 설계된 프로그래밍 언어이자 무료 소프트웨어 환경이다. R은 뉴질랜드 오클랜드 대학교의 Ross Ihaka와 Robert Gentleman에 의해 개발되었으며, 1995년에 처음 공개되었다.

- R은 S 프로그래밍 언어의 구현체이다
- 통계학자와 데이터 과학자 사이에서 데이터 분석, 통계 모델링, 시각화에 널리 사용된다
- R은 오픈 소스이며 방대한 패키지 생태계를 보유하고 있다 (CRAN에 19,000개 이상의 패키지)

### 1.2 GIS 통계학에 R을 사용하는 이유

R은 GIS 관련 통계 분석에 여러 가지 장점을 제공한다:

- **공간 데이터 처리**: `sf`, `sp`, `terra` 등 공간 데이터 관련 패키지 제공
- **통계 모델링**: 회귀분석, 시계열분석, 분류 등을 위한 내장 함수 제공
- **시각화**: `ggplot2`를 통한 출판 수준의 그래픽 및 지도 제작 가능
- **재현성**: 스크립트 기반 분석으로 재현 가능한 연구 보장
- **통합성**: Shapefile, GeoJSON, 래스터 데이터, 웹 매핑 서비스와 연동 가능

> **핵심 요점:** R은 전통적인 GIS 소프트웨어와 고급 통계 기법 간의 격차를 해소하여 공간 데이터 분석에 이상적인 도구이다.

## 2. R 및 RStudio 설치

### 2.1 R 설치

R은 CRAN(Comprehensive R Archive Network)에서 다운로드할 수 있다:
- 웹사이트: https://cran.r-project.org/
- Windows, macOS, Linux에서 사용 가능

### 2.2 RStudio IDE

RStudio는 R에서 가장 널리 사용되는 통합 개발 환경(IDE)이다:

- **Console**: R 명령어를 대화식으로 실행
- **Script Editor**: R 스크립트(.R 파일)를 작성하고 저장
- **Environment 패널**: 메모리에 있는 변수와 데이터를 확인
- **Plots/Help 패널**: 시각화 결과와 문서를 확인

### 2.3 기본 R 콘솔 연산

```r
# 기본 산술 연산
2 + 3          # 덧셈
10 - 4         # 뺄셈
5 * 6          # 곱셈
20 / 4         # 나눗셈
20 %/% 7       # 정수 나눗셈 (몫) -> 2
20 %% 7        # 나머지 연산 -> 6
6 ^ 2          # 거듭제곱 -> 36
6 ** 2         # 거듭제곱 (대체 표기) -> 36
```

### 2.4 도움말 사용

```r
help(mean)       # mean() 함수의 도움말 페이지 열기
?mean            # help()의 단축 표기
help(data.frame) # data.frame 함수의 도움말
```

> **핵심 요점:** `help()` 함수는 해당 함수가 어느 패키지에 속해 있는지 보여준다 (예: `mean {base}`는 `mean()`이 `base` 패키지에 있음을 의미한다).

## 3. 통계 프로그래밍 언어로서의 R

### 3.1 변수 할당

R은 변수 할당에 `<-` 연산자(권장) 또는 `=`를 사용한다:

```r
a <- 4       # 변수 a에 4를 할당
b <- 20      # 변수 b에 20을 할당
c <- a + b   # c = 24
```

**변수 이름 규칙:**
- 문자 또는 마침표(`.`)로 시작해야 한다
- 숫자, 밑줄, 특수 문자로 시작할 수 없다
- 대소문자를 구분한다 (`exam`과 `Exam`은 다른 변수)
- 공백 사용 불가; 대신 밑줄(`_`)을 사용한다

| 유효한 이름 | 유효하지 않은 이름 |
|-------------|-------------------|
| `exam`      | `1exam`           |
| `.exam`     | `_exam`           |
| `e1axm`     | `$exam`           |
| `e_xam`     | `ex a`            |

### 3.2 할당 연산자 상세

```r
A <- 2       # <-를 사용한 할당
B = 10       # =를 사용한 할당 (작동하지만 <-가 권장됨)
C = D <- 5   # C와 D 모두 5가 됨

# 중요한 차이: <-는 =보다 우선순위가 높다
# 함수 인수 내에서는 명명된 매개변수에 =를 사용해야 한다
sum(x <- 1)  # x가 전역 환경에 1로 할당됨
sum(y = 1)   # y는 명명된 인수이며, 전역적으로 할당되지 않음
```

> **핵심 요점:** R에서는 `=`보다 `<-`가 할당에 권장된다. `<-`는 연산자 우선순위가 더 높고, `=`는 함수 인수 지정용으로 예약되어 있기 때문이다.

### 3.3 R의 데이터 유형

R에는 여러 가지 기본 데이터 유형이 있다:

| 유형 | 설명 | 예시 |
|------|------|------|
| `numeric` | 실수 | `3.14`, `42` |
| `integer` | 정수 | `1L`, `100L` |
| `character` | 문자열 | `"Hello"`, `'World'` |
| `logical` | 논리값 | `TRUE`, `FALSE` |
| `factor` | 범주형 데이터 | `factor(c("A", "B"))` |
| `complex` | 복소수 | `3+2i` |

```r
# 유형 확인
class(42)           # "numeric"
class(TRUE)         # "logical"
class("Hello")      # "character"
class(factor("A"))  # "factor"

typeof(1:10)        # "integer"
```

## 4. GIS와 통계학

### 4.1 GIS란?

지리정보시스템(GIS, Geographic Information System)은 공간 및 지리 데이터를 수집, 관리, 분석하기 위한 프레임워크이다. GIS는 다음을 통합한다:

- **공간 데이터**: 위치 기반 정보 (좌표, 경계)
- **속성 데이터**: 공간 객체와 연관된 특성
- **통계 기법**: 지리 데이터에 적용되는 분석 방법

### 4.2 GIS에서의 통계 기법

이 강의에서는 지리 데이터에 일반적으로 사용되는 여러 통계 기법을 다룬다:

1. **기술 통계**: 공간 데이터 분포의 요약
2. **데이터 시각화**: `ggplot2`를 활용한 지도 및 차트 제작
3. **회귀분석**: 지리 변수 간 관계 모델링
4. **시계열분석**: 공간 데이터의 시간적 패턴 분석
5. **로지스틱 회귀**: 이진 공간 결과의 분류

### 4.3 GIS 통계를 위한 주요 R 패키지

| 패키지 | 용도 |
|--------|------|
| `ggplot2` | 데이터 시각화 및 지도 제작 |
| `dplyr` | 데이터 조작 및 변환 |
| `sf` | 공간 데이터를 위한 Simple Features |
| `mapview` | 대화형 지도 시각화 |
| `choroplethr` | 단계구분도(Choropleth map) 생성 |
| `tidyverse` | 데이터 과학 패키지 모음 |
